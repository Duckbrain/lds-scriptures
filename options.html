You can probably tell there is not a lot here. This is a release of some updates that I started before my mission, and have put many more hours into since then. I don't think I will be finishing it. Please check out the Contribute section to learn more about what is going on with development.

Catalog

Select a language from the dropdown and click "Refresh Catalog" to begin. Then expand folders and check the boxes to download books you will want access to. Sorry, I don't currently have a way to let you have several languages at once.

    Help

    Hey everyone, I know it's been a while since an update, and this is a huge change. I completely rewrote the code because Chrome made changes so the old code would not work right away. You will now have access to a lot more content, but it will be a little harder to do.

    If you are just looking to get back to using the scriptures, click Catalog and make sure it is on the language you want, and Reload the Catalog. Click on folders to expand to the books. Check the ones you want and they will download. Do not close the tab while downloads are going. I do not have the system fully stable yet.

    I know of a few things that are likely to cause stability issues. So to avoid issues, please take note of the following.

    • Refreshing the catalog erases all content.
    • I added a Download Visible to help with downloading large volumes of books, but it is far from stable. Do not use for large numbers of books.
    • Do not open other tabs of the app, or click on anything in the settings while downloads are going on.
    • There are experimental features that are hidden. To enable them click or type developerOn() in the console window Ctrl+Shift+J or Cmd+Shift+J.

    I realize that there wil probably be some issues with this release, so please email me with any problems you have.

    • Jonathan Duck
